How to Make Slay the Spire 2 Mods (Beginner Guide)(1)

Created2026-05-10|Updated2026-05-10
|Post Views:

Slay the Spire 2 Mod 製作 SOP ( 1 ) 準備工具

簡介

Slay the Spire 2 《殺戮尖塔2》是一款卡牌遊戲
此系列文章會一步一步引導讀者
完成《Slay the Spire 2》Mod 製作

環境資訊

作業系統：Windows 11

準備工具

1. Godot：遊戲影擎(開源免費)

2. BaseLib：開發模組工具(開源免費)

3. Visual Studio：代碼編輯器 (Community免費版)

4. Spine2D : 2D動畫編輯 (付費軟體)

1. Godot（遊戲引擎）

《殺戮尖塔2》使用 Godot 遊戲影擎開發。

製作Mod時我們需要用遊戲引擎來

  • 測試 Mod 是否正常運作
  • 調整 Mod 在遊戲場景內的效果

版本資訊

  • 使用版本：Godot_v4.5.1-stable_mono_win64
  • v4.5.1：引擎版本號
  • mono：支援 C# 的版本

下載位置：
https://github.com/godotengine/godot/releases

需要下載：

Godot_v4.5.1-stable_mono_win64.zip

2. BaseLib（模組工具）

BaseLib 是由社群開發 用來簡化 Mod 開發流程 的 函式庫。
可以簡單理解為 快速開發 Mod 的工具

下載位置：

-https://github.com/Alchyr/BaseLib-StS2/releases

3. Visual Studio（代碼編輯器）

《殺戮尖塔2》的 Mod 使用的程式語言為 C#

安裝方式

下載位置：

https://visualstudio.microsoft.com/downloads/

找到 Community 點選 

免費下載
下載完後
使用 Visual Studio Installer 安裝

Visaul studio 2022

必選項目

  • C#（.NET 9）

注意

須要安裝.Net C#才能 編譯 製作好的Mod

4. Spine2D（2D 動畫工具）

Spine2D 用來製作角色動畫與骨骼系統。

Slay the Spire 2 採用 2D 骨骼動畫系統
想要做 會動的角色 的 Mod 才需要這個軟體

使用方式

  1. 準備角色圖片（拆圖 / 分層）
  2. 匯入 Spine2D 並建立圖片資源
  3. 設計骨架（Skeleton）
  4. 綁定圖片（Slots / Attachments）
  5. 設計動畫（Animation Timeline）
  6. 匯出並導入遊戲使用

版本資訊

  • Spine 4.x（與 godot 4.X 相容版本）

官方網站

https://esotericsoftware.com/spine-download

Spine2D 為商業軟體，可以透過官方網站購買或使用試用版。
